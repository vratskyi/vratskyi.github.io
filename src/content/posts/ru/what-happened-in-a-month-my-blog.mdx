## Ну, и я не забыл обновить свой сайт.



**Обновленный:**
 

    1. Обновленный UI
	2. Обновленный файл стилей
	3. Исправлена ошибка, при которой <code>@tailwindcss/typography </code> не был применен к блогу
	4. Добавлена автоматическая сортировка сообщений по дате, от новых к старым
	5. Удален ненужный код из файла input.css
    6. Добавлен пакет astro-compress для сжатия файлов во время сборки проекта, настройки по умолчанию: <code> CSS: true, HTML: false, Image: true, JavaScript: true, SVG: false </code>



**Что я планирую сделать в будущих обновлениях:**
 

 1. Добавить пагинацию на страницу блога
 2. Интеграция i18n, перевод сайта на другой язык
 3. Добавить генерацию sitemap.xml
 4. Может быть, добавить hedless cms
